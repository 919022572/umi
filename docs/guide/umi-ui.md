# Introduce Umi UI

umi 项目的本地研发工作台。

## ✨ 特性

* 项目管理。集中式管理本地项目。
* 配置管理。umi / bigfish 常用项目配置。
* 任务管理。集成启动、构建、测试、代码规范检查、重新安装依赖等常用操作。

## 🔨 使用

```bash
$ umi ui
```

详见：[Umi UI 视频介绍](https://www.bilibili.com/video/av66178967)。
